Notes:

  • The daily profits page will only count and analyze all trades closed on that day.
  • If you have bought some coins and transferred them off Binance please delete that order to keep the results correct.
  • Before calculating profits you should remove all buy orders that you have made and then withdrawn the amount so the profits get calculated correctly.
  • If you want to recalculate profits delete all profits first using the Tools menu
