Welcome fellow trader!


Head over to the Tools page to get started. Remember it is still a work in progress. Please report any issues you have.


Getting Started:

  • STEP 1: Tools -> Imports -> Excel import
  • STEP 2: Tools -> Trades -> Combine trades (This will add all partial trades together.)
  • STEP 3: Tools -> Profits -> Calculate profits
  • STEP 4: Check out your profits on the Profit page

Notes:

  • All trade time are in UTC time. The date format used "yyyy-MM-dd HH:mm:ss"
  • The daily profits page will only count and analyze all trades closed on that day
  • Deleting one profit entry will also delete the trades associated with this profit.
  • All tables are sorted by trade time.
  • If you get an "Error page" while calculating profits try to match some trades manually and try again.
  • Total profits on the Profits All page are calculated based on the current USD values where as the daily total profit values are based on the market historical closing values on that day.
