Welcome fellow trader!


Head over to the Tools page to get started. Remember it is still a work in progress.


Getting Started:

  • STEP 1: Tools -> Excel Imports -> Excel import (the import takes up to a minute depending how many lines your spreadsheet has)
  • STEP 2: Tools -> Trades -> Combine trades (this will add all partial trades together. Can take a while).
  • STEP 3: Tools -> Profits -> Calculate profits (You should repeat this a few times)
  • STEP 4: Check out your profits

Notes:

  • The daily profits page will only count and analyze all trades closed on that day.
  • If you want to recalculate profits delete all profits first using the Tools menu
  • Deleting one profit entry will also delete the trades associated with this profit.
  • All tables are sorted by trade time.
  • If some sell and buy orders are not properly paired, just delete them and import the whole spreadsheet again. Only trades unique will be imported again.
  • You might have to click "calculate profits" more than once to get all trades profits calculated.
