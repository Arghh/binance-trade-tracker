Welcome fellow trader!


Head over to the Tools page to get started. Remember it is still a work in progress. Please report any issues you have.


Getting Started:

  • STEP 1: Tools -> Excel Imports -> Excel import (This takes up to a minute depending how many rows the TradeHistory has)
  • STEP 2: Tools -> Trades -> Combine trades (This will add all partial trades together. This can take a while)
  • STEP 3: Tools -> Profits -> Calculate profits (You should repeat this a few times)
  • STEP 4: Check out your profits

Notes:

  • All trade time are UTC GMT. The date format used "yyyy-MM-dd HH:mm:ss"
  • The daily profits page will only count and analyze all trades closed on that day
  • Deleting one profit entry will also delete the trades associated with this profit.
  • All tables are sorted by trade time
  • If some sell and buy orders are not properly paired, just delete them and import the whole spreadsheet again. Only unique trades will be imported again
  • You might have to click "calculate profits" more than once to get all trades profits calculated
